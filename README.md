# Spring / Angular / Crud Project

## Angular : Frond-End
To build Docker
```bash
$ cd front
$ docker build -t angular-crud -f Dockerfile .
```

To run Angular Docker End
```bash
$ docker-compose up
```

## Spring : Back-End
To build Docker
```bash
$ cd back
$ docker build -t spring-crud -f Dockerfile .
```
To run Spring Docker End
```bash
$ docker-compose up
```
